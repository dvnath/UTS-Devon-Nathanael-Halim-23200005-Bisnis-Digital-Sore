Rental Kostum Cosplay



Halo semuanya! Saya saat ini sedang mencoba membuat bisnis kecil kecilan, yaitu dengan membuka rentalan kostum untuk cosplay secara online. Nah saya mempromosikan rentalan saya melalui aplikasi Instagram dan Whatsapp. Pengunjung sekalian dapat mencarinya di Instagram dengan nama Mozzax Cosrent.


Kenapa saya memilih untuk membuka rentalan, karena saya melihat ada peluang dalam trend cosplay yang akan semakin berkembang nantinya. Sebelum saya memulai bisnis ini, saya mengamati dan mendata kostum mana yang akan laku untuk dirental. Tidak hanya itu saya juga memperhitungkan modal dan keuntungan yang akan didapat nanti.

Saya menyediakan beberapa macam variasi kostum cosplay yang bertemakan "Anime". Saya membeli kostum kostum tersebut langsung import dari pabrik di China. Jadi Pengunjung sekalian dapat mencoba juga untuk membuka rentalan kostum cosplay.
